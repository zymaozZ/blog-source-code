* 8种基本数据类型

    1. 整形

        > 1 byte = 8 bit

        | 类型  | 存储需求 | 取值范围                 |
        | ----- | :------: | ------------------------ |
        | int   |  4 byte  | -2^(7 * 4) ~ 2^(7 * 4)-1 |
        | short |  2 byte  | -2^(7 * 2) ~ 2^(7 * 2)-1 |
        | long  |  8 byte  | -2^(7 * 8) ~ 2^(7 * 8)-1 |
        | byte  |  1 byte  | -2^7 ~ 2^7-1             |

    2. 浮点类型

        | 类型                             | 表头   | 取值范围 |
        | -------------------------------- | ------ | -------- |
        | float                            | 4 byte |          |
        | double (数值精度是 float 的两倍) | 8 byte |          |
